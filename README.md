# Online Flower Shop
A modern e-commerce platform dedicated to fresh flowers, built with .NET and Flutter.

## Key Features
- **Product Catalog:** Browse flowers by categories (Weddings, Birthdays, etc.).
- **Shopping Cart:** Seamlessly add products and manage orders.
- **Cross-platform:** Available on Web and Mobile (Flutter).
- **Payment Integration:** Supports local payment methods.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript, Bootstrap, Flutter.
- **Backend:** ASP.NET Core Web API.
- **Database:** Microsoft SQL Server.
